Fazit: Die Lücke

Kein existierendes Programm kombiniert alles: Richtige Notennotation + Farben für Dynamik + Micro-Pitch + Micro-Timing + Live MIDI-Controller + Audio-Erkennung.

DAWs (Bitwig, Reaper, Ableton) haben die beste Expression-Kontrolle, aber zeigen nur Piano Roll, keine Notation.
Notationsprogramme (Dorico, MuseScore) zeigen schöne Noten, haben aber kaum Micro-Expression.

→ MusiAI könnte genau diese Lücke füllen: Notation + Expression + Live Control in einem.

6. Internes Datenformat-Konzept (MusiAI JSON)

Statt MusicXML könnte MusiAI ein eigenes JSON-Format nutzen, das alles direkt lesbar speichert:

{
  "tempo": 100,
  "divisions": 480,
  "time_signature": [4, 4],

  "notes": [
    {
      "pitch":       "C4",
      "cents":       0,          // exakt
      "velocity":    80,         // mf
      "duration_pct": 100,       // exakte Viertelnote
      "start_beat":  1.0
    },
    {
      "pitch":       "E4",
      "cents":       +4,         // 4 Cent höher
      "velocity":    100,        // f
      "duration_pct": 101,       // 1% länger
      "start_beat":  2.0
    },
    {
      "pitch":       "G4",
      "cents":       -10,        // 10 Cent tiefer (blue note)
      "velocity":    50,         // p
      "duration_pct": 97,        // 3% kürzer
      "start_beat":  3.0
    },
    {
      "pitch":       "C5",
      "cents":       +15,        // 15 Cent höher (expressiv)
      "velocity":    112,        // ff
      "duration_pct": 105,       // 5% länger (betont)
      "start_beat":  4.0
    }
  ]
}
